# Task Manager App

Mobile application prototype developed with React Native and Expo.

## Features
- Login and Register
- Task creation and management
- Animated UI
- Dark/Light mode
- Local storage using AsyncStorage

## Tech Stack
- React Native
- Expo
- TypeScript
- Context API
- AsyncStorage

## Architecture
The app follows a modular structure separating UI, logic, and data:

/components  
/screens  
/context  
/services  
/hooks  
/navigation  
/theme  

## Security considerations
- Input validation
- Local data persistence control
- Session management

## Installation

```bash
npm install
npx expo start
